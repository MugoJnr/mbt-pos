FIRST TIME SETUP (do this once)
────────────────────────────────
1. Make sure you have internet access
2. Double-click INSTALL.bat
   - If asked "Do you want to allow this app to make changes?" → click YES
   - Wait 1-3 minutes while dependencies download
   - You will see "Installation complete!" when done

3. Double-click RUN ME.bat to start the app
   - The setup wizard will appear on first run
   - Enter your shop name, create an admin password, configure printer

DAILY USE
──────────
- Double-click "RUN ME.bat" to start MBT POS
- Or create a shortcut to "RUN ME.bat" on your Desktop

REQUIREMENTS
─────────────
- Windows 10 or Windows 11
- Python 3.10 or newer (https://python.org — tick "Add Python to PATH")
- Internet connection (for Telegram features and license activation)
- At least 500 MB free disk space

LOGIN (default credentials)
────────────────────────────
- Username:  admin
- Password:  admin123
  (Change this immediately in Admin → Users)

TELEGRAM & AUTO REPORTS
────────────────────────
1. Open Telegram on your phone
2. Search @mbt_admin1_bot and tap Start
3. Send any message (e.g. "Hello")
4. In MBT POS: Settings → Telegram → click "Connect My Telegram"
5. Enable "Send daily sales report" in Settings (on by default after setup)
   Reports send when internet returns, then every 4 hours while online

M-PESA (each shop uses its own Till)
────────────────────────────────────
Settings → M-Pesa: enter your Till or Paybill number.
At checkout, choose M-Pesa and confirm when the customer has paid.
No customer personal data is stored.

EXPORT FOLDER
──────────────
Sales reports are saved to:
  C:\Users\[YourName]\Desktop\MBT POS Exports\
